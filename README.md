# Maitrey Vivek Phatak — Portfolio (GitHub Pages)

## Quick start (no build tools needed)
This site is plain HTML/CSS/JS. Works on GitHub Pages instantly.

### 1) Create a repo
**Option A (recommended):**
- Create a repository named: `YOURUSERNAME.github.io`

**Option B:**
- Create a repo named anything (e.g., `portfolio`) and enable Pages from the `main` branch.

### 2) Add files
Upload these files to the repo root:
- `index.html`
- `styles.css`
- `script.js`

### 3) Enable GitHub Pages
Repo → **Settings** → **Pages**
- Source: **Deploy from a branch**
- Branch: `main`
- Folder: `/ (root)`

Your site will be at:
- `https://YOURUSERNAME.github.io` (Option A)
- `https://YOURUSERNAME.github.io/REPO` (Option B)

## Customize
- Replace GitHub/LinkedIn/email links in `index.html`
- Add your real project links inside `script.js` (look for `github` and `live`)
